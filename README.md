# Inventory Management System

A Console-Based Inventory Management System developed using Python and MySQL. The project implements Object-Oriented Programming (OOP) concepts and MySQL database connectivity to manage products and inventory records efficiently.

## Features

* Add, Search, Update, and Delete Products
* Increase and Decrease Stock Quantity
* Low Stock Monitoring
* Highest Price Product Reporting
* Total Inventory Value Calculation
* Menu-Driven Interface
* MySQL Database Integration

## Technologies Used

* Python
* MySQL
* SQL
* Object-Oriented Programming (OOP)
* MySQL Connector

## Project Structure

* Product.py
* category.py
* supplier.py
* transaction.py
* inventory.py
* main.py
* db_connection.py
* inventory_db.sql
